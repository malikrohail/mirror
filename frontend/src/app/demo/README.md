# Demo Pages

These pages are a design sandbox used to brainstorm and iterate on UI wireframes with Claude Code. When stuck on a UX decision — layout, component styling, interaction pattern — I'd spin up a demo page here to quickly prototype options side-by-side and pick the best one.

Each subdirectory is a self-contained wireframe experiment (accessible at `/demo/<name>` in dev). They are not part of the production app.
