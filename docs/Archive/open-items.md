## LLM Prompt for Persona Description Generation

The short persona descriptions (e.g., "A 60-year-old with macular degeneration who uses high zoom and needs strong contrast") are generated by Claude during persona creation (Stage 1 of the LLM pipeline). The prompt and schema are defined in:

- **Prompt**: `backend/app/llm/prompts.py`
- **Response schema**: `backend/app/llm/schemas.py`
- **Persona engine**: `backend/app/core/persona_engine.py`
